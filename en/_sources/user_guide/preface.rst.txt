Preface
==========

PAMS is a python-based Platform for Artificial Market Simulations.
This package inherit the concept of `plham <https://plham.github.io/>`_ .
But, corresponding to the usage of deep learning, this package decide to employ python-based one for more extended usages.

This document consist of user guide and api references.
Moreover, the original repository also has many examples to implement many of artificial market simulation models.
